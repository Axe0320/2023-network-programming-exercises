# C Programming Exercises (2023 Spring Semester)

**ネットワークプログラミング演習** の課題プログラムです。
千葉工業大学 2年次 2023年前期に実施した演習で作成した C 言語プログラムをまとめています。

---

## リポジトリ構成

```
.
├── 00_warmup/                    ウォーミングアップ
├── 01_arrays-and-functions/      第1〜3週：配列と関数
├── 02_pointers-files-strings/    第4〜6週：ポインタ・ファイルI/O・文字列
```

## ビルド方法

GCC でコンパイルできます。

```bash
gcc -o output source.c
./output
```

`math.h` を使うプログラム（`1-6-1_deviation.c`、`1-6-2_deviation-en.c`）は `-lm` が必要です。

```bash
gcc -o output source.c -lm
```
